## Installation

1.  Clone the repo

    ```sh
    git clone https://github.com/manc1n1/sqlalchemy-challenge.git
    ```

2.  Change directories to `sqlalchemy-challenge`

    ```sh
    cd sqlalchemy-challenge
    ```

3.  Run `app.py`

    ```sh
    python app.py
    ```

## Usage

Open your browser of choice and enter `http://127.0.0.1:5000`
